# 音乐APP

## 项目功能

一个简单的音乐APP,实现了用户登录,注册,搜索,听歌,会员歌曲,添加歌单。接口使用网友提供的酷狗音乐API。先将数据获取到后端然后将数据清洗后再发送到客户端。项目客户端使用Java开发,并通过jetpack来搭建项目的整体框架.后端使用flask,msyql来管理并存储数据。

## 技术栈

- Jetpack

  搭建整体架构和页面导航

- Retrofit

  网络传输
